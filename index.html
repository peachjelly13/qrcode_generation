Making your own QR code using Python
Step-1: pip install qrcode
Step-2: open a text editor and make a python file
Step-3: import qrcode as qr (alias - optional)
Step-4: use qr.make(" insert here the link you want to make a qr of")
Example: qr.make("https://home.cern/science/physics")
Step-5: use the save function(" insert here the name you want to give the qr code and the way you want to save it as")
Example: code_ = qr.make("https://home.cern/science/physics")
code_.save("cern-website.png")
You have finally made a QRcode you can now generate a qr for your own website
